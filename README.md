# Git Commands

- Add All files 
```
git add .

```

- Commit All added files to git branch 
```
git commit -m "<Your commit message>"

```

- Push all commit changes to origin (git url)
```
git push

```

- Create New Branch 
```
git checkout -b <Your Branch Name>

```

- Changing the branches 
```
git checkout <Branch Name>

```
